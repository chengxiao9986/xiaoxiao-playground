## Key Sentences

**1.** "Adds a notion-id property to pages so that future imports can skip pages that have already been imported."
- 为页面添加一个 notion-id 属性，以便将来导入时可以跳过已经导入过的页面。
- **so that** 引导目的状语从句；**have already been imported** 现在完成时被动语态

**2.** "If any Notion syntax is not supported the static values will be saved instead."
- 如果不支持某些 Notion 语法，则会保存静态值。
- **if... will...** 第一条件句；**instead** 替代

**3.** "Leave as 'cover' if you don't have conflicts with existing properties."
- 如果与现有属性无冲突，请保留为"cover"。
- 祈使句 + if 条件从句

**4.** "Attachments will be saved according to your vault's attachment folder settings."
- 附件将根据你的库的附件文件夹设置进行保存。
- **according to** = 根据

# PM Email - Test Failure Report

> Source: Email to PM about integration test failures

## Translation

**Original (Chinese):**
由于最近的API调整，我们发现之前的integration test有几个case会fail。我们已经identify了root cause，是因为response schema变了，我们的parsing logic没有handle新的field。我们会在这周五之前fix这个issue，然后重新run所有的regression tests确保没有其他问题。

**Professional Email Version:**

"Due to recent API changes, we've discovered that several integration test cases are now failing. We have identified the root cause: the response schema has been updated, but our parsing logic doesn't handle the new fields. We will fix this issue by Friday and re-run all regression tests to ensure there are no other problems."

**Alternative (More Detailed):**

"Following the recent API adjustments, we've noticed that a few of our integration test cases have started failing. We've identified the root cause - the response schema has changed, and our current parsing logic isn't handling the newly added fields. We plan to address this issue before the end of this week (by Friday) and will subsequently run a full regression test suite to verify that no other issues exist."

## Key Sentences (Email Translation)

**1.** "Due to recent API changes, we've discovered that several integration test cases are now failing."
- 由于最近的API调整，我们发现有几个集成测试用例失败了
- **Due to** introduces the cause; **we've discovered that** is more professional than "we found"; **are now failing** (present continuous) emphasizes current state

**2.** "We have identified the root cause: the response schema has changed, and our current parsing logic isn't handling the newly added fields."
- 我们已经找到了根本原因：响应结构发生了变化，而我们当前的解析逻辑没有处理新增的字段
- **Colon (:)** used to introduce the specific explanation; **has changed** (present perfect) indicates recent change with current impact; **isn't handling** shows ongoing issue

**3.** "We will fix this issue by Friday and re-run all regression tests to ensure there are no other problems."
- 我们会在周五之前修复这个问题，并重新运行所有回归测试以确保没有其他问题
- **by Friday** = deadline indication; **re-run** = run again; **to ensure** expresses purpose
